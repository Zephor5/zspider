English Documentation
=====================

ZSpider is a self-hosted crawling platform for content monitoring and news aggregation.

This documentation is organized into three sections:

- ``guides``: development workflow and modernization planning
- ``architecture``: system shape, runtime behavior, and internal messaging
- ``reference``: data model and field references
